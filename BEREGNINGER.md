 ## Beregningsverifikation

  ---

  ## Priser

  | Pakke | 1 stk. | 2 stk. |
  |-------|--------|--------|
  | Basis | 7.999 kr. | 14.999 kr. |
  | Plus | 9.999 kr. | 18.999 kr. |
  | Premium | 11.999 kr. | 22.999 kr. |
  | Genopladelig | +750 kr./stk. | +1.500 kr. |

  ---

  ## Tilskud (2025-satser)

  | Tilskud | 1 øre | 2 ører |
  |---------|-------|--------|
  | Offentligt tilskud | 4.458 kr. | 6.888 kr. |
  | Ældre Sagen | 750 kr. | 1.500 kr. |
  | Sygeforsikring gr. 1-2 | max 1.500 kr. | max 3.000 kr. |
  | Sygeforsikring gr. 5 | max 1.000 kr. | max 2.000 kr. |
  | Sygeforsikring basis | max 500 kr. | max 1.000 kr. |
  | Helbredstillæg | 0-85% af rest | 0-85% af rest |

  ---

  ## Beregningsrækkefølge

  1. **Basispris + genopladelig** = Total
  2. **- Ældre Sagen rabat** (hvis valgt)
  3. **- Offentligt tilskud** (hvis valgt)
  4. **- Sygeforsikring** (50% af rest, max pr. øre)
  5. **- Helbredstillæg** (X% af rest)
  6. **= Egenbetaling**

  ## Testeksempler

  ### Eksempel 1: Premium 2 stk., kun offentligt tilskud

  | Post | Beløb |
  |------|-------|
  | Premium 2 stk. | 22.999 kr. |
  | Genopladelig | +1.500 kr. |
  | **Total** | **24.499 kr.** |
  | Offentligt tilskud | -6.888 kr. |
  | **Egenbetaling** | **17.611 kr.** |
  | Besparelse | 6.888 kr. (28%) |

  ---

  ### Eksempel 2: Premium 2 stk., alle tilskud, 85% helbredstillæg

  | Post | Beløb |
  |------|-------|
  | Premium 2 stk. | 22.999 kr. |
  | Genopladelig | +1.500 kr. |
  | **Total** | **24.499 kr.** |
  | Ældre Sagen | -1.500 kr. |
  | = | 22.999 kr. |
  | Offentligt tilskud | -6.888 kr. |
  | = | 16.111 kr. |
  | Sygeforsikring (50% = 8.056, max 3.000) | -3.000 kr. |
  | = | 13.111 kr. |
  | Helbredstillæg (85%) | -11.144 kr. |
  | **Egenbetaling** | **1.967 kr.** |
  | Besparelse | 22.532 kr. (92%) |

  ## Kilder

  - Offentligt tilskud 2025: [Heimdalls](https://heimdalls.dk/offentligt-tilskud/)
  - Sygeforsikring danmark: 50% af egenbetaling, max beløb pr. øre
  - Helbredstillæg: For pensionister/førtidspensionister, op til 85%
